# DSA Gemini Coach 🚀

An AI-powered DSA learning platform built using JavaScript and Gemini AI to help users understand data structures and algorithms interactively.

## Features ✨

* AI-powered DSA explanations
* Step-by-step problem-solving hints
* Time complexity discussions
* Beginner-friendly guidance
* Covers arrays, linked lists, trees, graphs, recursion, and DP
* Focuses on logic instead of directly revealing answers

## Tech Stack 🛠️

* JavaScript
* Gemini AI API
* HTML5
* CSS3

## Purpose 🎯

This project was created to improve DSA learning through conversational AI and guided problem-solving.

## Installation ⚙️

```bash
git clone https://github.com/Jharwal77/dsa-gemini-coach.git
cd dsa-gemini-coach
```

## Run Locally 💻

Open `index.html` in your browser
OR use Live Server in VS Code.
